# Description du module : Privilege 

Notre module et la pour aider les dévlopper C++ qui sont sous Windows à devlopper leur projet 
Je sais à quelle point cela peut-être compliquer à développer les programme qui gère les privilège 
tell que les élévation, les vérification des droit actuel, la modification ou supression de droit, 
getion des droit acl etc...

C'est pour cela que ce module et née dans la library Torixyna, c'est pour simplifier un maximum 
les dévlopper qui gère cela en C++ sous windows. Je suis ravie d'aider à la création de vôtre projet via 
ma library. 😉
